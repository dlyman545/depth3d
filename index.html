▸ Deploy Python Backend to Railway
1

Create a GitHub repo

Push app.py, requirements.txt, Procfile, and railway.toml to a new GitHub repository.

git init
git add app.py requirements.txt Procfile railway.toml
git commit -m "init"
git remote add origin https://github.com/YOU/depth3d-api.git
git push -u origin main
2

Deploy on Railway

Go to railway.app → New Project → Deploy from GitHub repo → select your repo. Railway auto-detects Python and installs dependencies.

3

Get your public URL

In Railway dashboard → Settings → Networking → Generate Domain. You'll get a URL like depth3d-api.up.railway.app.

4

Paste URL into this app

Switch to the Python engine above, paste your Railway URL, and the app will verify the connection automatically.
