 quickstart — local setup
STEP 01
Clone & setup

Run ./setup.sh (Mac/Linux) or setup_windows.bat to create a venv, install PyTorch for your CUDA version, and clone TripoSR.

STEP 02
Start server

Run ./start.sh (or start_windows.bat). The first launch downloads ~2 GB of model weights from HuggingFace.

STEP 03
Upload & convert

Drop any image with a clear subject. Best results: single object, plain background, 3/4 view angle.

STEP 04
Preview & export

The 3D mesh previews in-browser with WebGL. Download as GLB, OBJ, PLY, or STL for use in Blender, Unity, or a 3D printer.
